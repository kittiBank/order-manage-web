# Order Management Web Application

 This web application for order management system built with Next.js 16. This application provides a complete solution for managing customer orders with features like infinite scrolling, filtering, sorting, and user authentication.


### Features

- **User Authentication**: Complete auth system with login, registration, and password recovery
- **Order Management**: Create, view, and manage orders with detailed information
- **Advanced Filtering**: Filter orders by customer ID and status
- **Sorting Options**: Sort orders by date or total amount in ascending/descending order
- **Infinite Scrolling**: Smooth pagination with cursor-based infinite scroll
- **Protected Routes**: Secure pages that require authentication
- **Responsive Design**: Mobile-friendly interface built with Tailwind CSS

## Tech Stack

### Frontend
- **Next.js 16**: React framework with App Router
- **React 19**: UI library
- **TypeScript**: Type safety
- **Tailwind CSS 4**: CSS framework
- **TanStack React Query**: Data fetching and state management
- **SweetAlert2:  Beautiful alert modals

### Backend
- **Next.js API Routes** - RESTful API endpoints
- **Mock Data Storage** - In-memory data storage (easily replaceable with a real database)

## How to Run

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kittiBank/order-manage-web
   cd order-manage-web
   ```

2. **Install dependencies**
   ```bash
   npm install

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
